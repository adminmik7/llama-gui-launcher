# Llama.cpp Model Launcher

GUI-приложение для запуска llama-server на tkinter.

## Изменения

- **Раздельные KV-cache**: единый параметр `--cache-type-k/--cache-type-v` заменён на два отдельных поля `KV-cache K` и `KV-cache V`
- **Авто-определение хоста**: поле Host теперь автоматически подставляет локальный IP машины (вместо `192.168.3.151`)
- **Пустой порт по умолчанию**: поле Port теперь пустое по умолчанию
- **Потоки CPU**: значение по умолчанию берётся из `os.cpu_count()`
- **CPU MoE**: поле переименовано из "MoE экспертов" в "CPU MoE", соответствует `--n-cpu-moe`
- **Сохранение/загрузка конфига**: `cache` разделён на `cache_k` и `cache_v`
- **Раздельные Batch/UBatch**: единое `batch_size` разделено на `--batch-size` и `--ubatch-size`, оба по умолчанию 512
- **mmproj по умолчанию**: выбор mmproj файла теперь фильтруется по `*.gguf`
